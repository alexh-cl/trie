# Trie
HS Information Retrieval WS16/17 - Assignment 2



Naive implementation of a wildcard query search (e.g. col\*r) based on a Trie.



**Usage:**

```
./wildcard arg1
    arg1: INPUT - filename of a text file containing a word list
```
